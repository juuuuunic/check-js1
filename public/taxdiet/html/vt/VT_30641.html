부동산임대공급가액명세서 작성을 위해 아래 질문에 입력을 부탁드립니다.

임대현황

동 층 호 면적(㎡) 상호(성명)

※ 작성 시 참고

  • 동, 층, 호수 작성방법
  • 임대기간 등 작성방법

  • 계약건별로 각 임차인이 사용하는 동, 층, 호 기재

  • 동일 건물 내 여러 층을 하나의 보증금/월세로 일괄 계약하는 경우에는 모든 층(호)을 아래와 같이 기재

    • 예시1) 지하1층을 임대하는 경우: B1
    • 예시2) 지상1층, 지상2층, 지상3층을 임대하는 경우
    • 예시3) 201호, 202호, 203호, 205호 임대하는 경우: 201-203, 205
    • 예시4) 지하1층, 지상1층을 임대하는 경우: B1, 1

    • ※ 모든 층(호)의 면적(공용면적 포함)을 합산하여 기재

  • 과세기간 중 보증금/월세 변동 있는 경우

    • 계약갱신일 : 2017년 4월 1일
    • 종전 계약조건 : 보증금 5,000만원, 월세 50만원
    • 변경 계약조건 : 보증금 7,000만원, 월세 70만원
    입주일 갱신일 퇴거일 보증금 월세 특이사항
    2017.01.01 - 2017.03.31 50,000,000 500,000 종전 계약조건 입력
    2017.04.01 2017.04.01 2017.06.30 70,000,000 700,000 변경 계약조건 입력

    * 갱신 계약조건 입력 시 입주일은 갱신일, 갱신일은 계약변경일, 퇴거일은 과세기간 종료일(또는 퇴거일)을 각각 기재하셔야 합니다.

  • 과세기간 중 계약기간만 연장된 경우(보증금/월세 변동 없음)

    • 계약갱신일: 2017년 4월 1일
    • 변경 계약조건: 계약기간만 24개월(2년) 연장
    입주일 갱신일 퇴거일 보증금 월세 특이사항
    2017.01.01 - 2017.03.31 50,000,000 500,000 종전 계약조건 입력
    2017.04.01 2017.04.01 2017.06.30 50,000,000 500,000

    * 갱신 계약조건 입력 시 입주일은 갱신일, 갱신일은 계약변경일, 퇴거일은 과세기간 종료일(또는 퇴거일)을 각각 기재하셔야 합니다.

  • 갱신일

    • 과세기간 내에 계약기간 연장, 보증금, 월세 변동이 발생한 경우만 기재합니다.
      갱신일 기재 시 입주일 퇴거일은 반드시 기재하셔야 합니다.
