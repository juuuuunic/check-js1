  • Google 크롬 브라우저 팝업 차단 해제 안내

    Google의 크롬 브라우저를 사용하는 경우 팝업을 해제하는 방법은 다음과 같습니다.

     

    PC용 Goolge 크롬
    1. 브라우저 오른쪽에 메뉴 "아이콘"을 클릭합니다.
    2. "설정"을 클릭합니다.
    3. "고급 설정 표시"를 클릭합니다.
    4. 개인정보에서 "콘텐츠 설정" 버튼을 클릭합니다.
    5. "팝업"으로 스크롤한 후 모든 사이트에 "팝업 허용"을 클릭합니다. 
    6. 브라우저를 종료하고 다시 시작합니다.

     

    위 안내에 따라도 문제가 해결되지 않는 경우, 최신 버전의 브라우저로 업그레이드하는 것이 좋습니다.

  • Microsoft Internet Explorer 브라우저 팝업 차단 해제 안내

    Microsoft Internet Explorer 브라우저를 사용하는 경우 팝업 차단 해제하는 방법은 다음과 같습니다.

     

    1. Internet Explorer 상단 메뉴에서 "도구"를 클릭하고, 하단의 "인터넷 옵션"을 클릭합니다.

     

    2. "개인정보" 탭을 클릭합니다.
      ① '팝업 차단 사용'을 체크합니다.
      ② 팝업 차단 사용 옆 '설정'을 클릭 합니다.
      ③ 허용할 웹 사이트 주소 입력란에 www.nicesoho.co.kr을 입력합니다.
      ④ '추가'를 클릭합니다.
      ⑤ '닫기'를 클릭합니다.

     

    3. 브라우저를 새로고침(F5) 합니다.

     

    그래도 문제가 재현되면, 최신 버전으로 업그레이드 하세요.

  • Microsoft Internet Explorer 브라우저 쿠키 설정 안내

    Microsoft Internet Explorer 브라우저를 사용하는 경우 캐시, 쿠키 삭제하는 방법은 다음과 같습니다.

     

    1. Internet Explorer 상단 메뉴에서 "도구"를 클릭하고, 하단의 "인터넷 옵션"을 클릭합니다.

     

    2. 임시 인터넷 파일 및 쿠키를 삭제합니다.
      ① '일반' 탭을 선택합니다.
      ② '검색 기록' [삭제] 버튼을 클릭합니다.
      ③ '검색 기록 삭제' 창이 뜨면, 모든 체크박스에 체크를 합니다.
      ④ [삭제] 버튼을 클릭합니다.

     

    3. 브라우저를 닫았다가 다시 시작합니다.

     

    그래도 문제가 재현되면, 최신 버전으로 업그레이드 하세요.

  • 휴면 안내 메일 정책이란?

    나이스소호는 정보통신망 이용촉진 및 정보보호 등에 관한 법률(이하, 정보통신망법) 제29조 2항 및 동법 시행령 제16조에 의거하여, 1년 이상 당사 서비스를 이용하지 않는 회원의 개인정보를 보호하기 위하여 휴면 정책을 시행합니다.

     

    휴면계정으로 전환되면 나이스소호에서 제공하는 서비스를 제공 받으실 수 없으며, 향후 언제라도 서비스 재이용을 원하시면 로그인을 통해 재사용 전환이 가능합니다.

     

    감사합니다.

  • 회원탈퇴 방법을 알려주세요.

    당사에서 제공하는 서비스는 세무신고 자료로 활용이 가능한 자료이므로, 탈퇴 요청을 하시면 당사 직원이 별도 확인 전화를 드리고 있습니다.
    마이페이지에서 회원탈퇴 페이지에서 회원탈퇴로 인한 확인 전화 기간을 체크 후 탈퇴요청해 주세요.
    단순 ID변경을 원하시는 고객님께서는 고객센터로 문의해 주세요.

     

    탈퇴시 당사에서 제공하는 서비스와 관련된 데이터(카드이용데이터, 세금계산서 발행목록 등)는 일괄삭제되며,
    삭제된 데이터는 복구가 불가능하오니, 탈퇴 요청은 각별히 주의해 신청해 주세요.

    기타 문의사항은 고객센터 1588-5659번으로 문의해 주세요.

    서비스 이용약관이란?

    나이스소호는 회원님들의 권익을 보호하고, 회원들의 개인정보 보호에 만전을 기하고자, 관계법령의 규정을 반영하여 약관을 변경하였습니다.

    개인정보처리방침을 통하여 고객님께서 제공하는 개인정보가 어떠한 용도와 방식으로 이용되고 있으며,

  • 나이스소호 회원가입은 어떻게 하나요?

    나이스소호는 개인회원/사업자 회원/제휴카드 사업자 회원으로 가입할 수 있으며, 내국인만 가입이 가능합니다.

     

    개인회원 가입은
    ① 나이스소호 회원가입(개인회원)을 선택해 주세요.
    ② 아이디, 비밀번호,를 입력해 주세요.
      - 나이스소호 아이디는 6~12자의 영소문자, 숫자만 사용할 수 있어요.
      - 비밀번호는 8자 이상의 영문 대/소문자, 숫자, 특수문자를 함께 사용해야 합니다.
    ③ 이메일, 이름을 입력해 주세요.
      - 이메일, 이름은 정확하게 입력하지 않으면 서비스 제한이 있을 수 있습니다.
    ④ 휴대폰 번호를 입력하고 [인증번호 전송]을 선택해 주세요. 입력한 휴대폰 번호로 인증번호가 발송됩니다.
      - 발송된 인증번호는 10분 이내로 입력해 주세요. 혹시 인증번호가 도착하지 않았다면 [재발송]을 선택해 주세요.
      - 인증번호 입력 후 [인증하기]를 선택해 주세요.
    ⑤ [완료] 버튼을 선택하면, 개인 회원가입이 완료됩니다.

     

    사업자 회원/제휴카드 사업자 회원은
    ① 나이스소호 회원가입(사업자 회원)을 선택해 주세요.
    ② 사업자등록번호를 입력해 주세요.
      - 입력하신 사업자등번호로 제휴카드 신청하신 고객님은 ③ 내용에서 제휴 카드번호 확인 하실 수 있습니다.
    ③ 서비스 이용약관 및 개인정보처리방침에 대한 내용 확인 후 [동의]체크 하셔서 [다음]을 선택해 주세요
      - ②에서 입력하신 사업자등록번호로 제휴카드 신청하신 고객님은 하단 내용에 제휴 카드번호를 입력해 주세요.
    ④ 상호명, 대표자명, 사업장 전화번호를 입력해 주세요.
      - 상호명, 대표자명, 사업장 전화번호를 정확하게 입력하지 않으면 서비스 제한이 있을 수 있습니다.
    ⑤ 휴대폰 번호를 입력하고 [인증번호 전송]을 선택해 주세요. 입력한 휴대폰 번호로 인증번호가 발송됩니다.
      - 발송된 인증번호는 10분 이내로 입력해 주세요. 혹시 인증번호가 도착하지 않았다면 [재발송]을 선택해 주세요.
      - 인증번호 입력 후 [인증하기]를 선택해 주세요.
    ⑥ 사업장 주소는 [우편번호]를 선택 하셔서 작성해 주세요.
    ⑦ 아이디, 비밀번호,를 입력해 주세요.
      - 나이스소호 아이디는 6~12자의 영소문자, 숫자만 사용할 수 있어요.
      - 비밀번호는 8자 이상의 영문 대/소문자, 숫자, 특수문자를 함께 사용해야 합니다.
    ⑧ 이메일, 이름을 입력해 주세요.
      - 이메일, 이름은 정확하게 입력하지 않으면 서비스 제한이 있을 수 있습니다.
    ⑨ 휴대폰 번호를 입력하고 [인증번호 전송]을 선택해 주세요. 입력한 휴대폰 번호로 인증번호가 발송됩니다.
      - 발송된 인증번호는 10분 이내로 입력해 주세요. 혹시 인증번호가 도착하지 않았다면 [재발송]을 선택해 주세요.
      - 인증번호 입력 후 [인증하기]를 선택해 주세요.
    ⑩ [다음] 버튼을 선택해 주세요.
    ⑪ 세무주치의 서비스 동의를 선택해 주세요.
      - 담당 세무주치의를 확인해 주세요.
      - 담당 세무주치의를 변경 원하시면 [다른 세무주치의 보기]에서 선택해 주세요.
      - 세무주치의 서비스를 동의하거나 나중에 설정 체크를 해주세요. 나중에 설정하시면 가입 후 마이페이지에서 서비스 설정 가능합니다.
    ⑫ [완료] 버튼을 선택하시면 가입이 완료됩니다.
